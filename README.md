# Feedguard-for-Tiktok
FeedGuard is a browser extension that helps you clean up your TikTok Web feed.

Features:
- Skip ads
- Skip LIVE videos
- Filter hashtags
- Filter keywords
- Use blacklist and whitelist rules
- Pause and resume filtering
- View statistics in a dark mode dashboard
- Import and export settings as JSON

FeedGuard runs locally in your browser and stores settings using chrome.storage.local.
FeedGuard is not affiliated with, endorsed by, or sponsored by TikTok.


Installation
Download and extract the ZIP file.

open chrome://extension; for edge: edge://extensions/

Enable developer mode.

Load the extracted extension (Button on top)

Select the extracted folder FeedGuard_for_TikTok_v1_0
